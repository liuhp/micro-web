# micro-web
微前端

工程简介：

portal -- portal主应用

subapp1 -- 子应用1

service -- 服务端

#### 单独启动某个应用
```shell script
cd portal # 进入到要启动应用的工程根目录下
npm start # 启动项目
```
#### 一条命令启动所有应用和及服务端
```shell script
# micro_web根目录下执行
npm start 
```
